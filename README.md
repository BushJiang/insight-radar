# 智源雷达（InsightRadar）

智源雷达用于发现 GitHub 上最有价值的开源项目，帮助程序员持续了解值得关注、学习和使用的项目。

## 核心功能

- 采集 GitHub 开源项目信息
- 识别和评估项目价值信号
- 追踪项目热度、活跃度和发展趋势
- 使用智能体辅助分析项目质量与推荐理由

## 技术栈

- TypeScript
- Next.js 15
- Tailwind CSS + shadcn/ui
- Bun
- PostgreSQL
- Drizzle ORM
- Neon / Supabase
- Better Auth
- Mastra
- Vercel / EdgeOne Pages
- Milvus/Zilliz（按需引入）

## 目录结构

- `src/`：源代码
- `tests/`：测试套件
- `demo/`：独立可运行原型
- `resource/`：参考资料和外部原始资料
- `spec/`：项目规约
- `docs/`：项目文档
- `notes/`：团队成员笔记
- `chats/`：对话记录
- `.42cog/`：认知敏捷法文档
- `.42plugin/`：本地技能库

## 快速开始

```bash
bun install
bun dev
```
